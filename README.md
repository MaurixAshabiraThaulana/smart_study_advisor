# Smart Study Advisor

Sistem Cerdas berbasis Web menggunakan Python (Flask) dan HTML.

## Deskripsi
Smart Study Advisor adalah sistem cerdas berbasis rule-based expert system
yang memberikan rekomendasi strategi belajar mahasiswa berdasarkan:
- Gaya belajar
- Waktu luang
- Tingkat stres
- Nilai akademik

## Fitur
- Login & Register User
- Sistem Cerdas (Rule-Based)
- Penyimpanan Riwayat Rekomendasi
- Database SQLite
- User Interface berbasis Web

## Teknologi
- Python (Flask)
- HTML & CSS
- SQLite
- GitHub

```bash
git clone https://github.com/USERNAME/smart-study-advisor.git
